Luas Lingkaran

Diketahui = a. phi= 3,14
	    b. r = 14

Ditanya = Luas Lingkaran ?
Jawab = phi*r*r
      = 3,14*14*14
      = 615.44

Keliling Lingkaran

Diketahui = a. phi= 3,14
	  = b. r = 7

Ditanya = Keliling Lingkaran ?
Jawab = 2*phi*r
      = 2*3,14*7
      = 43.96
